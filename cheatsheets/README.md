# Cheatsheets

This folder contains my compact Linux topic cheatsheets.

## What will be here
- paths and navigation;
- file permissions;
- SSH;
- processes;
- systemd;
- networking;
- grep/find;
- pipes/redirection.

## Format
Each file should answer 3 questions:
1. What is it?
2. Which commands are used most often?
3. What are the common mistakes?

## Purpose
This is not a book summary and not a learning diary.
It is a quick-reference folder for revision.
