# TLCL Notes

This folder contains my short notes on the book *The Linux Command Line*.

## Goal
The goal is not to rewrite the whole book, but to capture:
- the main ideas of each chapter;
- important commands;
- my own examples;
- how the topic connects to Linux Upskill Challenge.

## Rules
- keep notes short;
- write in my own words;
- add 2–5 command examples of my own;
- mention where the topic was useful in practice
